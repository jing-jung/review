# 심플 To-Do 메모 앱

HTML, CSS, JavaScript로 구현된 간단하고 반응형인 To-Do 메모 애플리케이션입니다.
이 프로젝트는 웹 개발 입문자가 바이브 코딩 기법으로 기본적인 CRUD 기능을 이해하고, 브라우저의 localstorage API를 활용한 데이터 저장 바어법을 학습할 수 있도록 설계됐습니다.

## 기능
* **메모 추가:** 입력창에 할 일을 적고 '추가' 버튼이나 Enter 키를 눌러 목록에 추가할 수 있습니다.
* **메모 삭제:** 각 항목 우측의 '삭제' 버튼을 눌러 메모를 지울 수 있습니다.
* **데이터 유지:** LocalStorage를 활용하여 브라우저를 새로고침하거나 다시 열어도 기록된 메모가 유지됩니다.
* **반응형 디자인:** 모바일 및 데스크탑 환경에 맞게 레이아웃이 자연스럽게 조정됩니다.

## 파일 구성
* `index.html`: 앱의 기본 구조
* `style.css`: 시각적 스타일링 및 반응형 레이아웃 처리
* `script.js`: 메모 추가, 삭제 및 LocalStorage 데이터 관리 로직
