AI와 협업하며 아이디어를 즉시 프로토타입으로 만드는 새로운 개발 방식

⚡ 빠른 검증

🤝 개발 소통 강화

직접 만들어보며 개발 흐름을 이해하게 된다.

🧠 AI 활용 능력

단순 문서 요약을 넘어 실제 서비스를 만드는 경험.

30분 실전 따라하기

1

아이디어 정리

목적, 사용자, 핵심 기능, 참고 UI를 AI가 이해할 수 있게 설명한다. 

[목적] 무엇을 위한 앱인가?
[사용자] 누가 사용하는가?
[핵심 기능] 꼭 필요한 기능 3개
[참고 UI] 어떤 앱 느낌인가?
2

Claude에서 코드 생성

프롬프트를 입력하면 React 기반 앱 코드를 자동 생성한다.

“코드 전체를 하나의 파일로 만들어줘”를 추가하면 편하다.
3

Bolt.new에서 실행

생성된 코드를 붙여넣으면 실시간으로 앱이 실행된다.

4

반복 수정

원하는 변경사항을 자연어로 다시 설명하며 앱을 개선한다. 

버튼 색상을 파란색으로 바꿔줘.
완료율을 상단에 표시해줘.
팀원 필터 기능을 추가해줘.
5

배포 및 공유

생성된 앱을 URL로 배포하고 팀원과 바로 공유한다.

기획자를 위한 프롬프트 공식

[역할] 너는 숙련된 프론트엔드 개발자야.
[목표] 업무 체크리스트 앱을 만들어줘.
[기술] React + Tailwind CSS 사용
[UX] Notion 스타일의 깔끔한 UI
[조건] 파일 하나로 실행 가능하게
