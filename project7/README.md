# 초보자용 날씨 웹앱 (Weather Web App)

이 프로젝트는 초보자를 위한 간단한 날씨 정보 제공 웹 애플리케이션입니다. HTML, CSS, Vanilla JavaScript만을 사용하여 제작되었으며, 브라우저의 Geolocation API와 OpenWeather API를 연동하여 사용자의 현재 위치 기반 날씨를 보여줍니다.

## 🚀 주요 기능
- **현재 위치 감지**: 브라우저의 Geolocation API를 사용하여 위도, 경도를 자동으로 감지.
- **날씨 데이터 가져오기**: OpenWeather API (Current Weather Data)를 호출하여 실시간 날씨 데이터 표시.
- **상세 날씨 정보 제공**:
  - 기본 정보: 도시 이름, 현재 온도, 날씨 상태(설명 및 아이콘).
  - 추가 정보: 체감 온도, 습도, 풍속.
- **사용자 경험(UX) 최적화**: 
  - 다크 테마 기반의 모던한 2단 레이아웃 (640x400px 고정 크기).
  - 데이터 로딩 중 메시지 표시.
  - 날씨 데이터 로드 완료 시 API 입력 영역 자동 숨김 기능.
  - 각종 에러 처리 (API Key 미입력, 권한 거부, 잘못된 키 입력 등).

## 🛠️ 사용 기술
- **HTML5**: 웹 페이지 구조 작성
- **CSS3**: 플렉스박스(Flexbox) 기반의 반응형 스타일링 및 UI 디자인
- **JavaScript (ES6+)**: 
  - `fetch` API 및 `async/await`를 활용한 비동기 통신
  - DOM 조작 및 이벤트 리스너 추가

## 💻 실행 방법
1. 코드가 들어있는 폴더를 엽니다.
2. `index.html` 파일을 더블 클릭하여 크롬(Chrome)이나 엣지(Edge) 등의 웹 브라우저에서 실행합니다.
3. 화면에 표시된 입력창에 **OpenWeather API Key**를 입력합니다.
4. **"날씨 가져오기"** 버튼을 클릭합니다.
5. 브라우저 상단에서 **위치 정보 접근 권한**을 요청하는 팝업이 뜨면 **"허용"**을 선택합니다.
6. 로딩 후 예쁜 다크 테마 카드에 해당 위치의 날씨 정보가 표시되며 API 입력창은 사라집니다.

## 🔑 OpenWeather API Key 발급 방법
1. [OpenWeather 공식 웹사이트](https://openweathermap.org/)에 접속합니다.
2. 우측 상단의 **Sign in**을 클릭하고, 계정이 없다면 **Create an Account**를 눌러 회원가입을 합니다.
3. 로그인 후 화면 우측 상단의 프로필 이름을 클릭하고 **My API keys** 메뉴로 들어갑니다.
4. 화면에 보이는 긴 영문/숫자 조합의 문자열이 본인의 `API Key`입니다.
5. **(주의사항)** 새로 발급받은 API Key는 활성화되기까지 **최대 10~15분 정도 소요**될 수 있습니다. 활성화되기 전에 사용하면 `유효하지 않은 API Key` 에러가 발생할 수 있습니다.
